# npv

**npv** (*nimipäivä*) is a simple CLI for looking up finnish namedates. Made as a "toy" project to get to know [Hare programming language](https://harelang.org/).

The data itself lives inside *./namedates.ha*. Extracted and formatted into valid Hare data from https://nimipaivat.com/nimipaivakalenteri/
